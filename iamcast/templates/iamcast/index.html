{% trans "Manejo de la Información" %}

{% trans "El manejo de la información dentro de una agencia de casting debe ser tomado seriamente. Es muy común dejar anotaciones de nombres, telefonos y otros datos, tanto de agenciados, como de productoras y trabajos, en libretas, papeles y cuadernos." %}

{% trans "Hay varias desventajas en manejarse de esta forma: " %}

  • {% trans "Es complicado realizar busquedas dentro de la información almacenada de esta forma." %}
  • {% trans "La información no es accesible desde cualquier lugar." %}
  • {% trans "Lleva mucho tiempo reproducir/copiar la información." %}
  • {% trans "En equipos de trabajo, se torna complicada la colaboración." %}
  • {% trans "Muchas veces se trabaja con información desactualizada." %}

{% trans "La aplicación IamCast ofrece la posibilidad de guardar la información en forma centralizada y segura, de forma que la misma sea consistente y su acceso sea simple." %}

{% trans "¿Qué se puede guardar?" %}

{% trans "La información que se puede guardar en IamCast es la siguiente:" %}

  • {% trans "Perfil de sus agenciados (incluyendo fotos y videos de youtube)" %}
  • {% trans "Productoras" %}
  • {% trans "Trabajos y los perfiles buscados por cada trabajo" %}
  • {% trans "Postulaciones de sus agenciados a los trabajos publicados" %}
  • {% trans "Portfolio (por cada item del portfolio se puede asociar una foto y/o video de youtube)" %}

{% trans "Acceso a la información" %}

{% trans "Existen dos formas de acceder a la información:" %}

  • {% trans "Como administrador:" %}
    • {% trans "Solo pueden acceder los integrantes de la agencia (agenciadores)." %}
    • {% trans "Podrá administrar agenciados, productoras y trabajos, también se pueden administrar grupos, usuarios y permisos." %}
  • {% trans "Como usuario web o agenciado:" %}
    • {% trans "Desde la página web usted podrá captar agenciados. Los mismos podrán registrarse y completar su perfil online." %}
    • {% trans "Luego, desde el módulo de administración, usted podrá activar los agenciados en los cuales tenga interés." %}

{% trans "Procesos de búsqueda de perfiles" %}

{% trans "Una vez que usted tenga en su sistema agenciados, productoras y trabajos (cada trabajo con sus respectivos perfiles), usted podrá postular agenciados a cada perfil. " %}

{% trans "Para poder llevar a cabo esta tarea, en el módulo de administración dispondrá de un listado de agenciados en el cual podrá aplicar filtros por características físicas, habilidades, ubicación geográfica, fecha de nacimiento, etc." %}

{% trans "Luego de aplicar los filtros correspondientes, usted podrá seleccionar agenciados y postularlos a alguno de los perfiles." %}

{% trans "Una vez que haya terminado el proceso de postulación para cada perfil de un determinado trabajo, podrá realizar el envío de mail automático a la productora solicitante del trabajo." %}

{% trans "En dicho mail aparecerá información correspondiente al trabajo. Por cada perfil de dicho trabajo, se enviarán las postulaciones de agenciados. Por cada agenciado postulado se mostrará: foto(s), nombre, apellido y fecha de nacimiento." %}

{% trans "A medida que se avance en las distintas etapas de un trabajo, se podrá:" %}

  • {% trans "Agregar y quitar postulaciones." %}
  • {% trans "Modificar estado de postulaciones." %}
  • {% trans "Agregar eventos a trabajos y/o perfiles." %}
