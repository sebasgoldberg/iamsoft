{% trans "Trabajos" %}

{% trans "Ahora que haz ingresado algunos agenciados en tu sistema, podemos empezar a ver el módulo de trabajos en la administración de tu agencia. Antes vamos a hacer una breve explicación de algunos conceptos." %}

{% trans "Dentro del módulo de trabajo podemos encontrar los siguientes objetos:" %}

{% trans "Productora" %}
{% trans "Las productoras en general son los clientes de los trabajos que realizamos con nuestra agencia." %}
{% trans "Trabajo" %}
{% trans "Es cada trabajo a realizar por nuestra agencia. Cada trabajo se realiza para alguna productora. Los componentes de un trabajo son:" %}
  • {% trans "Productora (el cliente del trabajo)" %}
  • {% trans "Eventos del trabajo (los eventos asociados al trabajo: casting, callback, prueba de ropa, realización)" %}
  • {% trans "Perfiles buscados (Los distintos perfiles buscados)" %}
{% trans "Perfil Buscado" %}
{% trans "Un perfil buscado es un componente de un trabajo. Un trabajo puede tener asociado varios perfiles buscados, por ejemplo, para la filmación de una pelicula (el trabajo) podemos buscar protagonistas y extras (los perfiles buscados). Protagonistas es un perfil buscado y Extras es otro perfil buscado. Ambos pertenecen al trabajo película. Cada uno tendrá asociadas determinadas características, cache, etc. Un perfil buscado está compuesto por" %}
  • {% trans "Pertenece a un trabajo" %}
  • {% trans "Eventos del perfil buscado (similar a los eventos del trabajo pero en este caso son eventos específicos del perfil)" %}
  • {% trans "Postulaciones (las postulaciones de los agenciados hechas a los distintos perfiles)" %}
{% trans "Postulación" %}
{% trans "Una postulación es la acción de proponer un agenciado para un perfil buscado de un determinado trabajo. Cada postulación tendrá un determinado estado que tiene como objetivo indicar por ejemplo si el agenciado se encuentra solamente postulado, si ya fué seleccionado, si realizó el trabajo o si ya sele pagó el trabajo." %}

{% trans "Ahora que ya tenemos una idea de cómo es la organización de los distintos objetos en el módulo de trabajo, vamos a ver más en detalle cada uno de los mismos." %}
